Project Space

Project gemaakt in leerjaar 3: GameDesign Space

In dit project heb ik een 3D first-person shooter game ontwikkeld met enkele parkour-elementen. Ik heb me sterk gefocust op geluid en verlichting gedurende het hele spel.

Tijdens dit project heb ik geleerd hoe ik Scriptable Objects in Unity kan gebruiken en heb ik meer kennis opgedaan over 3D-animaties, evenals een dieper begrip van geluids- en verlichtingen.

Project Turtle Dash

Project gemaakt in leerjaar 2: Groep Project Arcade

Ik heb aan dit arcadespel gewerkt met een team van vijf — drie tekenaars en twee developers. Mijn rol bestond uit het coderen van het startscherm en het programmeren van de speler- en healt systemen.

Door dit project heb ik een dieper begrip van Unity gekregen, waarbij ik heb geleerd hoe 2D-animaties werken, hoe je omgaat met schaalvergroting en hoe je de UI effectief kunt gebruiken. We hadden ook uitstekend teamwerk en sterke communicatie gedurende het hele project, wat het ontwikkelproces soepel en efficiënt maakte.

Project Piano Madeness

Project gemaakt in leerjaar 1: Unity Piano

Deze piano is gemaakt tijdens mijn eerste jaar op school. Het was een goede leerervaring voor mij, omdat het mijn eerste kennismaking was met het werken met geluid in Unity, waar ik de basis van audio Unity heb geleerd.

De piano is volledig bespeelbaar met zowel het toetsenbord als de muis. De toetsen voor het spelen zijn: Caps Lock, A, S, D, F, G, H, J, K, L, Q, W, R, T, Y, I, O.

About

Hoi! Ik ben Samantha, een student softwareontwikkeling met een passie voor het maken van leuke games. Hoewel gameontwikkeling mijn voornaamste focus is, vind ik het ook leuk om webprojecten te maken, zoals deze portfolio. Mijn enthousiasme voor gamen heeft me ertoe gebracht om een carrière in deze industrie na te streven, en ik ben ernaar uit om meer te leren en mijn vaardigheden te verbeteren, zodat ik games kan ontwikkelen die iedereen leuk vindt. :)

In mijn vrije tijd blijf ik graag actief door te zwemmen (met de reddingsbrigade) en te fitnessen. Ik houd ook van lezen en naar muziek luisteren om gefocust te blijven en van mijn tijd te genieten.
