Projects

Cat Game

3D CatGame Escape the house!

This project was created with the idea that you play as a cat trying to escape an old grandma’s house. Your goal is to find the keys, breaking objects along the way to uncover their hiding spots.

Be careful to avoid the grandma. If she gets mad, she might kill you! The game features numerous interactive elements and a dark twist that adds depth and intrigue.

Skill developd: C#, 3D Game Development

Groeps project Vietnyam

Vietnyam

I created this project with two other programmers. We made a cat game where you play as a cat and have to defeat other cats. After defeating an enemy, you’re presented with a choice menu to select a power-up. Unfortunately, the player’s visuals are not completely finished.

I personally animated the main menu, created the pause menu with particles and hover effects, and developed one of the two boss fights, as shown in the images. The boss has three phases: the first phase uses only laser eyes, the second phase only shoots bullets, and in the third phase, both are active.

Skill developd: 2D animations, C#, Parical system unity, Teamwroking

Project fish

Fish Game!

When I created this game, it was preparation for an exam. I had 3 weeks to develop a fish game.

I managed to create a fully functional fish game. The fish can eat, grow, and drop coins. The larger the fish, the more coins they produce, which you can use to buy even more fish. I also added a Grim Reaper who appears occasionally and tries to kill the fish.

Skill developd: C#, 2D Game Development

Project Space

GameDesign Space

In this project, I developed a 3D first-person shooter game with some parkour elements. I focused heavily on sound design and lighting throughout the game.

During this project, I learned how to use Scriptable Objects in Unity and gained deeper knowledge of 3D animations, as well as deeper understanding of sound and lighting techniques.

Skill developd: Game design, 3D sounds, C#

Project Turtle Dash

Group Project Arcade

I worked on this arcade game with a team of five —three artists and two developers. My role involved coding the start screen, as well as programming the player and health systems.

Through this project, I gained a deeper understanding of Unity, learning how 2D animations work, how to handle scaling, and how to effectively use the UI. We also had excellent teamwork and strong communication throughout the project, which made the development process smooth and efficient.

Skill developd: 2D Unity, C#, Teamwroking

Project Piano Madeness

Unity Piano

This piano was created during my first year of school. It was a significant learning experience for me as its my first experience working with sound in Unity, where I learned the basics of audio design.

The piano is fully playable using either the keyboard or mouse. The keys for playing are: Caps Lock, A, S, D, F, G, H, J, K, L, Q, W, R, T, Y, I, O.
